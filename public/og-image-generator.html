Method 1: Online Converter (Easiest)

  1. Go to CloudConvert or Convertio
  2. Upload public/og-image.svg
  3. Set output size to 1200x630px
  4. Convert and download
  5. Save as public/og-image.png

Method 2: Using Browser (Quick)

  1. Open public/og-image.svg in Chrome/Firefox
  2. Right-click on the image
  3. Select "Save Image As..."
  4. Save as PNG (some browsers support this directly)
  5. Or take a screenshot at 1200x630 resolution

Method 3: Using Figma/Canva (Best Quality)

  1. Open Figma or Canva
  2. Create a new design with dimensions 1200 x 630 px
  3. Import the SVG file or recreate the design
  4. Export as PNG (2x for retina quality)
  5. Save as public/og-image.png

Method 4: Install ImageMagick (Terminal)

# Install ImageMagick via Homebrew
brew install imagemagick

# Convert SVG to PNG
magick public/og-image.svg -resize 1200x630 public/og-image.png

# Or use rsvg-convert
brew install librsvg
rsvg-convert -w 1200 -h 630 public/og-image.svg -o public/og-image.png
